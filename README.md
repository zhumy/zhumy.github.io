## Research Areas

| Area | Topics |
| --- | --- |
| Spatial Audio & Binaural Rendering | HRTF-based virtual audio, 3D sound rendering, multizone reproduction, room acoustics |
| Speech & Audio Enhancement | Voice activity detection, speech enhancement, residual echo suppression, packet loss concealment, quality assessment |
| Microphone Array Processing | Beamforming, DOA estimation, TDOA-based localization, acoustic vector sensors, spherical microphone arrays |
| AI for Computational Audition | Deep learning for audio, end-to-end speech models, keyword spotting, speaker verification, auditory scene understanding |
| Audio Coding & MDCT | Frequency estimation in MDCT domain, audio codec algorithms, multimedia IC design |
| GPU & FPGA Acceleration | High-performance computing, graphics rasterization, real-time rendering, hardware-aware signal processing |
